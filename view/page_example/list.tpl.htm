[add] 新增Page區塊範本

這裡可以讓你建立Page區塊的範本
當你要建立Page區塊的時候，會從這裡挑一筆啟用的當範本

<{*將php傳來的變數，轉成固定名稱的變數，讓section迴圈使用*}> <{php}>$this->_tpl_vars['rows'] = $this->_tpl_vars['pages'];<{/php}> <{if count($rows) <= 0 }> 無資料 <{else}> <{section name=num01 loop=$rows}> <{/section}>

<{$rows[num01].name|default:" "}> <{$rows[num01].description|default:" "}> <{if $rows[num01].enable == '1'}> 啟用 <{else}> 停用 <{/if}>
