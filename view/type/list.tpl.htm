[add] 新增欄位型態
這裡存放的欄位，是mysql程序引數的型態
讓使用者可以在建立程序引數的時候，用下拉的方式去選擇型態
<{*將php傳來的變數，轉成固定名稱的變數，讓section迴圈使用*}> <{php}>$this->_tpl_vars['rows'] = $this->_tpl_vars['types'];<{/php}> <{if count($rows) <= 0 }> 無資料 <{else}> <{section name=num01 loop=$rows}> <{/section}>
型態名稱 修改 刪除
<{$rows[num01].name|cat:" "}>
修改
刪除
<{/if}>
